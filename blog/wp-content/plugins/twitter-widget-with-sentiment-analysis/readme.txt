Twitter Widget with Tweet Reach: Shows latest tweets for any keyword or, your handle along with Twitter Reach. 

== Description ==
Shows latest tweets for any keyword, hashtag or, your handle along with Twitter Reach. 
Twitter Widget with Tweet Reach is best twitter sidebar plugin widget you will find for WordPress. It 
easy to use, easy to install and has all the features you needs to show your twitter activity, feeds 
off. Twitter Widget with Tweet Reach is a sidebar widget that offers you the possibility to show your 
lastest in your WordPress web site. 

= How it works = 
On load, Plugin sends a request to <a href="https://www.socialappshq.com">SocialAppsHQ</a> and it looks up latest tweets based on your keyword. We then compute twitter 
reach and trends based on results. Twitter Reach shows how far your tweets reach?.

If an email is provided in your settings, you can contact us to create an account for free so that you can look at raw data as well as all kind of fancy analysis on top of it - it's a free service.

== Installation ==

1. Upload the `twitter-widget` folder to the `/wp-content/plugins/` directory
2. Activate the twitter widget through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Twitter Widget` menu that appears in your admin menu (left)
